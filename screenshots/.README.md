# Screenshots

This folder contains proof of the working Windows domain environment.

## Images

### 01-server-domain-admin.png
Shows the Windows Server acting as the Domain Controller.
The command `whoami` confirms the session is running as `lab\administrator`.

### 02-windows11-domain-login.png
Shows the Windows 11 client logged into the domain using the domain user.
The `whoami` command confirms the session as `lab\testuser`.

### 03-active-directory-user.png
Shows the Active Directory Users and Computers console.
The domain user `Test User` is visible inside the domain.

### 04-shared-folder-contents.png ###
Shows the shared folder on the server (`C:\Shared`) containing a test file.  
This folder is accessible from the domain-joined Windows 11 client.

### 05-backup-folder-contents.png ### 
Shows the backup directory (`C:\Backups\Shared_Backup`) after the backup script execution.  
The file from the shared folder has been successfully copied.

### 06-backup-script-execution.png ###  
Shows the command prompt output while running the backup script (`backup.bat`).  
The output confirms that files were copied from `C:\Shared` to the backup directory.

### 07-intern-security-log-denied.png ###
Shows the user intern1 being denied access to the Security event log
