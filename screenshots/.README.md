# Screenshots

This folder contains proof of the working Windows domain environment.

## Images

### 01-server-domain-admin.png
Shows the Windows Server acting as the Domain Controller.
The command `whoami` confirms the session is running as `lab\administrator`.

### 02-windows11-domain-login.png
Shows the Windows 11 client logged into the domain using the domain user.
The `whoami` command confirms the session as `lab\testuser`.

### 03-active-directory-user.png
Shows the Active Directory Users and Computers console.
The domain user `Test User` is visible inside the domain.
